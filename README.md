# CRM

A simple Customer Relationship Management (CRM) application for managing a client database and their records. The project was built in the Node.js environment using the Handlebars template engine and a simple CSS stylesheet.

### Requirements
* Node.js
* npm package manager

### Setup Instructions

1. Clone the repository:
`git clone https://github.com/Arthemisxx/CRM.git`

2. Navigate to the project directory:
`cd CRM`

3. Install dependencies:
`npm install`

4. Run the application:
`node index.js`

The application will be available in your browser at http://localhost:3000.
